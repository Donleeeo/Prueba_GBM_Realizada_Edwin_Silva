Instrucciones de Uso

Archivo Comprimido: Dentro de este archivo comprimido encontrará el archivo Ejercicio 2 - Formula1.ipynb. Este archivo debe ser ejecutado en plataformas como Google Colab o Jupyter Lab para facilitar la depuración del código.

Pruebas Unitarias

Ubicación y Ejecución de Pruebas: Dentro de la carpeta comprimida encontrará tres archivos suficientes para probar el funcionamiento del código. Al ejecutar el archivo Ejercicio 2 - Formula1.ipynb, se desplegará una casilla en la consola para probar su funcionamiento interactivamente.

NOTA: El ejercicio funcionará introduciendo en la consola la ruta del archivo .txt sin comillas.

Manejo de Entradas y Salidas:

Se incluyen tres archivos adicionales para realizar las pruebas unitarias:

datos_entrada.txt: Contiene las entradas esperadas con los resultados de input y output esperado.
datos_prueba.txt: Posee datos en un formato no esperado.
datos_prueba_en_blanco.txt: Este archivo está vacío y sirve para evaluar cómo se comporta el código ante la falta de datos.

Nota adicional: Recuerde que el código permite ingresar la ruta del archivo desde la consola sin comillas. Esto se realiza al ejecutar la primera celda para probar su ejecución de manera interactiva.
